## Exercise
### API-Exercise I
#### Task

Using flask to develop a API for sum, minus, multiply, and divide two parameters.

**URI**: /sum, /minus, /multiply, /divide

**Parameters**: value1, value2

**Example**:

Request:

```
http://127.0.0.1/sum?value1=1&value2=1
```

Response:

```
2
```
